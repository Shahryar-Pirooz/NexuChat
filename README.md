# NexusChat
**NexusChat** is a modern, lightweight, and real-time chat application built with Golang and powered by NATS for seamless messaging. Designed with simplicity and efficiency in mind, NexusChat allows users to join a chatroom, send and receive messages instantly, and view active participants using a sleek command-line interface (CLI).
